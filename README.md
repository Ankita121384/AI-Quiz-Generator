# 🧠 AI Quiz Generator

An AI-powered quiz generator that creates custom MCQ quizzes on any topic instantly!

## 🌐 Live Demo
👉 [Click here to try the app](https://ai-quiz-generator-asuafjerozszhn9drvs2a6.streamlit.app)

## ✨ Features
- 📚 Generate quiz on ANY topic
- 🎯 Choose difficulty: Easy, Medium, Hard
- 🔢 Choose 3–10 questions
- 🤖 AI generates unique questions every time
- ✅ Instant score with correct answers
- 💡 Explanation for every answer

## 🛠️ Tech Stack
- **Python** — Core programming language
- **Streamlit** — Web interface
- **Groq AI (LLaMA 3)** — Quiz generation engine
- **python-dotenv** — Environment variable management
